## 1. ISR 1941 라우터 설정 (Gateway & NAT)

라우터는 외부 공인 IP를 내부 사설 IP로 변환하고, 하단 장비들에 IP를 자동으로 할당하는 역할을 수행합니다.

```bash
! --- 인터페이스 설정 (외부 Gi0/0, 내부 Gi0/1) ---
interface GigabitEthernet0/0
 description WAN_CONNECTION
 ip address dhcp             # ISP로부터 IP 자동 수신
 ip nat outside              # 외부 NAT 인터페이스로 지정
 no shutdown

interface GigabitEthernet0/1
 description LAN_CONNECTION
 ip address 192.168.1.1 255.255.255.0
 ip nat inside               # 내부 NAT 인터페이스로 지정
 no shutdown

! --- NAT(PAT) 설정: 내부 PC들이 하나의 공인 IP를 공유 ---
access-list 1 permit 192.168.1.0 0.0.0.255
ip nat inside source list 1 interface GigabitEthernet0/0 overload

! --- DHCP 설정: PC에 IP/G.W/DNS 자동 전달 ---
ip dhcp pool INTERNAL_POOL
 network 192.168.1.0 255.255.255.0
 default-router 192.168.1.1
 dns-server 8.8.8.8

```

## 2. Catalyst 2960 스위치 설정 (L2 Switching & Trunk)

스위치는 내부 단말들을 연결하며, 상단 라우터와의 통신 통로(Uplink)를 관리합니다.

```bash
! --- 관리용 IP 및 게이트웨이 설정 ---
interface Vlan1
 ip address 192.168.1.2 255.255.255.0
 no shutdown

ip default-gateway 192.168.1.1

! --- 포트 설정: 사용자 포트(Access) 및 업링크(Trunk) ---
interface range FastEthernet0/1 - 24
 switchport mode access
 switchport access vlan 1
 spanning-tree portfast      # 단말기 연결 속도 향상

interface GigabitEthernet0/1
 description UPLINK_TO_ROUTER
 switchport mode trunk       # 확장성을 고려한 트렁크 설정

```

## 3. PC 설정 (End Device)

패킷트레이서의 PC 장비에서 수행해야 할 설정입니다.

1. **IP Configuration:** PC를 클릭하고 `Desktop` 탭의 `IP Configuration`으로 이동합니다.
2. **DHCP 선택:** 'Static' 대신 **'DHCP'** 라디오 버튼을 클릭합니다.
3. **확인:** 라우터로부터 `192.168.1.x` 대역의 IP와 게이트웨이(`192.168.1.1`)를 정상적으로 받아오는지 확인합니다.

---

### 실무 체크리스트

* **물리적 연결:** 라우터의 `Gi0/1`과 스위치의 `Gi0/1`이 연결되었는지 확인하십시오.
* **인터페이스 상태:** `show ip interface brief` 명령어로 모든 관련 포트가 **Status: up, Protocol: up** 인지 확인하십시오.
* **NAT 확인:** `show ip nat translations` 명령어로 PC의 통신 데이터가 변환되고 있는지 확인하십시오.
